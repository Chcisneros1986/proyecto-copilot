Formas de representar un grafo

Lista de Adyacencia

Utiliza una lista para cada nodo, donde se almacenan los nodos adyacentes. Es eficiente en espacio para grafos dispersos.

{
A: [B, C],
B: [A, D],
C: [A, D, E],
D: [B, C],
E: [C]
}

Matriz de Adyacencia

Utiliza una matriz donde las filas y columnas representan nodos. Un valor 1 indica la presencia de una arista. Es eficiente para consultar aristas.

   A B C D E
A 0 1 1 0 0
B 1 0 0 1 0
C 1 0 0 1 1
D 0 1 1 0 0
E 0 0 1 0 0
